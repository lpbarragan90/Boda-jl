
# Mesas J&L — PWA (iPhone listo) con Sync opcional

1) Sube todos los archivos a la raíz de tu repositorio y activa GitHub Pages (main + /root).
2) Abre la URL de Pages en Safari (iPhone). Si la tenías instalada, bórrala y vuelve a instalar.
3) Para sincronizar entre iPhones:
   - Abre index.html y rellena FIREBASE_CONFIG con tu proyecto Firebase.
   - (Opcional) habilita Auth anónima y reglas `request.auth != null`.
4) Puedes añadir invitados en la barra derecha (botón "Agregar invitado") o importar un JSON exportado.
